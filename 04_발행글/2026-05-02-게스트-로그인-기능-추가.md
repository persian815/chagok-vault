# 게스트 로그인 기능 추가

로그인 없이도 앱을 둘러볼 수 있게 됐다.

---

## 오늘 한 것

"게스트로 둘러보기" 버튼을 만들었다. 로그인 화면 아래에 회색으로 조용하게 붙어있는 버튼인데, 누르면 Supabase의 익명 인증(Anonymous Auth)으로 임시 계정이 자동 생성되면서 앱 안으로 들어간다.

게스트는 기본적으로 **조회만 가능**하고 데이터를 건드리는 기능은 막혔다.

- 지식 추가/삭제 ❌
- 프롬프트 편집 ❌
- 구매내역 수동 추가 ❌
- Storage 동기화 ❌

대신 AI 질문은 자유롭게 할 수 있고, 오늘 추가로 **구매내역 이미지 분석도 허용**했다. 처음엔 막아뒀는데, 게스트가 자기 영수증을 찍어서 써보고 싶을 수 있잖아. 게스트가 저장한 데이터는 익명 계정 ID로 자동 분리되니까 오너 데이터와 섞일 걱정도 없다.

## 막혔던 부분

처음 버튼을 눌렀을 때 "게스트 로그인에 실패했습니다."라는 메시지만 뜨고 원인을 몰랐다. 에러를 숨겨버렸던 거였다. 실제 에러를 보니 `anonymous_provider_disabled` — Supabase 대시보드에서 Anonymous 인증을 켜지 않은 것이었다. Authentication → Providers → Anonymous 토글 하나 켜니까 바로 됐다.

## 다음에 할 것

게스트가 앱을 써보고 마음에 들면 정식 계정으로 전환할 수 있으면 좋겠다. Supabase에 익명 계정 → 이메일 계정 업그레이드 기능이 있으니 나중에 연결해볼 수 있을 것 같다.
