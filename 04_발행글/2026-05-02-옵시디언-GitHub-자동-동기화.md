# 옵시디언 + GitHub 자동 동기화 설정법

알뜰살뜰 앱의 블로그 탭은 `persian815/blog` 레포의 `posts/` 폴더에서 .md 파일을 읽는다. 옵시디언에서 글을 쓰고 GitHub에 자동으로 올라가게 설정하는 방법을 정리한다.

---

## 전제 조건

- 옵시디언 설치 완료
- `persian815/blog` 레포가 로컬에 clone되어 있어야 함

```bash
git clone https://github.com/persian815/blog.git ~/Documents/blog
```

---

## 1단계 — 옵시디언 Vault 설정

1. 옵시디언 실행
2. **Open folder as vault** 클릭
3. `~/Documents/blog` (또는 clone한 경로) 선택

이제 옵시디언이 blog 레포 전체를 Vault로 인식한다.

---

## 2단계 — Obsidian Git 플러그인 설치

1. Settings → **Community plugins**
2. "Turn on community plugins" 클릭 (처음이면)
3. Browse → **Obsidian Git** 검색
4. Install → Enable

---

## 3단계 — 플러그인 설정

Settings → Obsidian Git:

| 항목 | 권장값 | 설명 |
|---|---|---|
| Auto pull interval | `10` | 10분마다 GitHub에서 pull |
| Auto backup interval | `10` | 10분마다 자동 commit + push |
| Auto backup after file change | ✅ | 파일 수정 감지 후 백업 |
| Commit message | `post: {{date}} auto sync` | 자동 커밋 메시지 형식 |

---

## 4단계 — GitHub 인증

맥에서는 보통 키체인에 GitHub 자격증명이 저장되어 있어 별도 설정이 필요 없다. 만약 push 시 인증 오류가 나면:

```bash
git -C ~/Documents/blog config user.name "persian815"
git -C ~/Documents/blog config user.email "persian815@gmail.com"
```

GitHub Personal Access Token이 필요한 경우:
1. GitHub → Settings → Developer settings → Personal access tokens
2. `repo` 권한으로 토큰 생성
3. push 시 비밀번호 대신 토큰 입력

---

## 사용 흐름

```
옵시디언에서 posts/ 폴더에 .md 파일 작성
→ 10분 후 Obsidian Git이 자동 commit + push
→ 알뜰살뜰 앱 설정 → 블로그 탭에서 즉시 조회 가능
```

### 파일명 규칙

앱에서 날짜와 제목을 자동 파싱하려면 아래 형식을 지킨다:

```
YYYY-MM-DD-제목.md
```

예시:
- `2026-05-02-알뜰살뜰-개발기.md`
- `2026-05-10-장보기-팁.md`

---

## 즉시 발행하고 싶을 때

자동 sync를 기다리지 않고 바로 올리려면 Claude Code에서:

```
/skills-posts
```

변경된 .md 파일을 감지해서 즉시 commit + push한다.
