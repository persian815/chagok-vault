# /skills-log — 결정·삽질 옵시디언 기록

개발 중 중요한 결정이나 삽질이 생겼을 때 즉시 옵시디언에 기록한다.
세션 끝날 때 한꺼번에 정리하면 잊어버리니까, 생긴 즉시 실행한다.

---

## 실행 절차

### 1. 현재 날짜 확인

```bash
date +%Y-%m-%d
```

### 2. 오늘 개발일지 파일 확인

```bash
ls /Users/ilmare815/work/CHAGOK/02_프로젝트/[현재 프로젝트]/개발일지/YYYY-MM-DD-개발일지.md
```

파일이 없으면 템플릿(`03_템플릿/개발일지-템플릿.md`)을 복사해서 생성한다.

### 3. 기록 유형 판단

사용자가 전달한 내용이:

- **삽질·버그 해결**이면 → 개발일지 `막힌 것/해결` 섹션에 추가
  + `01_공통지식/삽질모음.md`에도 동일 내용 추가
- **설계 결정**이면 → 개발일지 `결정한 것` 섹션에 추가
  + 중요한 결정이면 `01_설계.md` ADR 섹션에도 추가
- **오늘 한 것**이면 → 개발일지 `한 것` 섹션에 추가

### 4. 파일 업데이트

해당 섹션에 내용을 추가한다.
기존 내용은 건드리지 않는다.

### 5. 완료 보고

```
✅ 기록 완료
파일: YYYY-MM-DD-개발일지.md
섹션: [추가된 섹션명]
```

---

## 사용 예시

```
/skills-log Cerebras llama3.1-8b 모델만 안정적으로 동작함. 다른 모델명은 404
/skills-log 오늘 게스트 로그인 기능 완성
/skills-log Vercel 요청 크기 제한 4.5MB → next.config.mjs에서 10mb로 변경
```

## 주의

- 현재 작업 중인 프로젝트 경로를 정확히 확인하고 기록한다
- 삽질은 반드시 `삽질모음.md`에도 추가 (다음 프로젝트에서 재활용)
- 커밋은 하지 않는다 (옵시디언 Git 플러그인이 자동 처리)
