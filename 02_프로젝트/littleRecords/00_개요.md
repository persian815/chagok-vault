# 알뜰살뜰 (Little Records)

> 차곡 브랜드 1호 프로젝트
> 쇼핑몰 주문 캡처 이미지를 AI가 파싱해주는 가계부 앱

---

## 한 줄 요약

캡처 이미지 하나 올리면 AI가 알아서 파싱해주는 부부 가계부

---

## 탄생 배경

신혼집 입주 후 9개 쇼핑몰에서 162건, 약 1,061만 원 구매.
엑셀로 수동 정리하다 v5까지 만들고 포기 → AI 파싱 앱 직접 제작.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js 14 + TypeScript |
| Styling | Tailwind + shadcn/ui |
| DB/Auth | Supabase |
| 이미지 AI | Gemini 2.5 Flash |
| 텍스트 AI | Cerebras llama3.1-8b |
| 배포 | Vercel |

---

## 주요 기능

1. **이미지 파싱** — 쇼핑몰 캡처 → 물건명·금액·날짜·쇼핑몰 자동 추출
2. **AI Q&A** — "이번 달 얼마 썼어?" 자연어 질문 가능
3. **지식베이스** — 브랜드 정보·예산 계획 .md 파일로 관리
4. **대시보드** — 카테고리별 도넛 차트, 월별 트렌드
5. **게스트 로그인** — 로그인 없이 체험 가능
6. **블로그 탭** — 옵시디언에서 쓴 글이 앱 안에서 바로 보임
