# dayOne

> 차곡 브랜드 0호 프로젝트 (개인용)
> 일일 기록 관리 앱 — 사진·메모·iCal을 한 곳에서

---

## 한 줄 요약

사진 찍으면 EXIF로 날짜·장소 자동 입력, 에피소드 기록하고 iCal로 내보내는 개인 일지 앱

---

## 탄생 배경

하루하루 기록을 남기고 싶은데 Apple 메모는 검색이 약하고 노션은 오버스펙.
사진 EXIF에서 날짜·좌표를 자동 파싱해서 기록 부담을 줄이고, 캘린더에서도 볼 수 있도록 ICS 출력 기능 직접 제작.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js 16 (App Router) + TypeScript |
| Styling | Tailwind CSS 4 + shadcn/ui |
| DB/Auth | Supabase PostgreSQL + Supabase Auth |
| Storage | Supabase Storage (이미지) |
| 배포 | Vercel |

---

## 주요 기능

1. **에피소드 기록** — ep 번호·날짜·지역·장소·여행명·시간·메모·좌표 저장
2. **EXIF 자동 파싱** — 사진 업로드 시 촬영일시·GPS 좌표 자동 입력
3. **ICS 생성** — 에피소드를 iCal 파일로 내보내기 (캘린더 앱 연동)
4. **히스토리** — 과거 기록 조회·수정·ICS 재다운로드
5. **이미지 업로드** — Supabase Storage에 사진 보관
