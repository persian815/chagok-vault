# 두리 (Duri)

> 차곡 브랜드 2호 프로젝트
> 성룡 & 주영의 결혼 준비를 함께 관리하는 커플 웨딩 플래너 앱

---

## 한 줄 요약

D-day 카운터 + 캘린더 + 타임라인 + 체크리스트가 하나로 합쳐진 커플 전용 결혼 준비 앱

---

## 탄생 배경

결혼 준비 할 일이 48개, 일정이 수십 개인데 노션은 두 사람이 실시간으로 쓰기 불편.
대시보드는 예쁜데 새로고침하면 체크가 날아감 → 실시간 동기화되는 앱 직접 제작.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js 14 + TypeScript |
| Styling | Tailwind + shadcn/ui |
| DB/Auth | Supabase |
| Realtime | Supabase Realtime |
| 배포 | Vercel |

---

## 주요 기능

1. **D-day 카운터** — 본식까지 남은 날 실시간 표시
2. **캘린더** — 월별 일정 등록·수정·삭제, 담당자 태그
3. **타임라인** — 6~9월 섹션별 진행률, 우선순위(🥇🥈🥉) 그룹화
4. **체크리스트** — 전체 진행률, 담당자 필터, 완료 토글
5. **실시간 동기화** — 성룡이 체크하면 주영 화면에 바로 반영
