# 차곡 (chogok)

> 차곡 브랜드 5호 프로젝트
> 알뜰살뜰 + dayOne + 두리를 하나로 통합한 부부 일상 기록 허브 앱

---

## 한 줄 요약

소비·에피소드·할일을 하나의 캘린더에서 차곡차곡 쌓는 부부용 통합 일상 기록 앱

---

## 탄생 배경

알뜰살뜰(소비), dayOne(에피소드), 두리(결혼준비)를 각각 쓰다 보니
홈 화면에 앱이 세 개가 되었고, 캘린더에서 세 도메인을 한눈에 보는 것이 불가능했다.
세 앱을 하나로 합쳐 부부의 일상 데이터를 차곡차곡 쌓는 허브를 만들기로 결정.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js 15 (App Router) + TypeScript |
| Styling | Tailwind CSS 4 + shadcn/ui |
| DB/Auth | Supabase PostgreSQL + Supabase Auth |
| Storage | Supabase Storage |
| Realtime | Supabase Realtime |
| 이미지 AI | Gemini 2.5 Flash |
| 텍스트 AI | Cerebras llama3.1-8b |
| 배포 | Vercel |
| PWA | next-pwa |

---

## 주요 기능

1. **통합 캘린더** — 에피소드·할일·소비를 3색 dot으로 월별 조망
2. **소비 탭** — 영수증 사진 → Gemini AI 파싱 + Cerebras 자연어 Q&A
3. **할일 탭** — 프로젝트별 체크리스트 + Supabase Realtime 동기화
4. **에피소드** — 사진 EXIF 파싱으로 날짜·장소 자동 입력 + ICS 생성
5. **FAB** — 어느 화면에서든 에피소드·소비·할일 빠르게 추가
6. **PWA** — iPhone 홈 화면 아이콘 설치 지원
