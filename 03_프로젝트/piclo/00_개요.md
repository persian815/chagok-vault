# 사진 정리 웹앱 (piclo)

> 차곡 브랜드 4호 프로젝트
> 핸드폰 사진을 AI로 자동 분류·정리해주는 웹앱

---

## 한 줄 요약

사진을 업로드하면 AI가 날짜·장소·내용 기반으로 앨범을 자동으로 만들어주는 모바일 최적화 웹앱

---

## 탄생 배경

핸드폰에 수천 장의 사진이 쌓였지만 정리할 엄두가 나지 않는 상황.
날짜별·장소별로 직접 분류하는 작업이 너무 번거로워,
AI가 대신 분류하고 앨범을 만들어 ZIP으로 내려받을 수 있는 서비스를 기획.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js 15 (App Router) + TypeScript |
| Styling | Tailwind CSS 4 + shadcn/ui |
| DB/Auth | Supabase PostgreSQL + Supabase Auth |
| Storage | Supabase Storage (사진 원본) |
| AI 분류 | Google Gemini 2.5 Flash Vision |
| EXIF 파싱 | exifr (브라우저 사이드) |
| HEIC 변환 | heic2any (브라우저 사이드) |
| PWA | manifest.json + Service Worker |
| 배포 | Vercel |

---

## 주요 기능

1. **사진 업로드** — 갤러리 다중 선택, HEIC 자동 변환, 진행률 표시
2. **AI 자동 분류** — Gemini Vision으로 카테고리(인물·음식·풍경·문서·스크린샷) + 태그 추출
3. **EXIF 기반 그룹핑** — 날짜·GPS 정보로 앨범 자동 생성
4. **앨범 뷰어** — 카드 그리드, 사진 상세 보기, 앨범명 수정
5. **다운로드** — 앨범 단위 ZIP 다운로드, 선택 사진 다운로드
6. **PWA** — 홈 화면 추가, 뷰어 오프라인 캐시
