## v0.1 — MVP (본식 전 완성 목표)

### 레이아웃 & 공통 UI
| 기능 | 상태 | 비고 |
|---|---|---|
| 진입 팝업 (오늘하루 안보기 + X 닫기) | ✅ | 배경 딤처리, localStorage |
| 배경음악 재생/정지 버튼 | ✅ | BgmContext, 팝업 닫기 시 자동 재생 |
| 위로가기 + 공유 버튼 (우측 하단 상시) | ✅ | |
| 골드 포인트 컬러 (#C4A882) | ✅ | CSS 변수 `--gold` |
| Playfair Display (영문 세리프) 폰트 | ✅ | Google Fonts |
| 스크롤 fade-in 애니메이션 | ✅ | Intersection Observer |
| WeddingConfig Context | ✅ | DB 설정을 컴포넌트 트리 전체에 공급 |

### 섹션
| 섹션 | 기능 | 상태 |
|---|---|---|
| S01 히어로 커버 | 전체화면 사진 + 세리프 타이포 + 날짜 | ✅ |
| S02 초대 문구 | 텍스트 + 전체너비 사진 | ✅ |
| S03 인물 소개 | 신랑/신부 2열 카드 (이름·양가부모·자녀순서) | ✅ |
| S04 웨딩 인터뷰 | 버튼 → 바텀시트 모달 (Q&A, 신랑🤵/신부👰 구분) | ✅ |
| S05 예식 안내 | 날짜·시간·장소 + 커플 사진 + 9월 달력 (19일 골드 원) | ✅ |
| S06 D-Day | 전체화면 사진 오버레이 + 💗 + D-XXX + 날짜 | ✅ |
| S07 우리의 이야기 | 지그재그 타임라인 (사진↔텍스트, 중앙 세로선+골드 점) | ✅ |
| S08 갤러리 | 3열 모자이크 그리드 → 라이트박스 (스와이프, n/8, X) | ✅ |
| S09 게스트스냅 | 서브페이지 링크 버튼 | ✅ |
| S10 포토부스·주차·답례품 | 3탭 네비 (사진 + 안내 텍스트) | ✅ |
| S11 오시는 길 | 장소+주소 + Google Maps iframe + [티맵\|카카오내비\|네이버지도] 버튼 | ✅ |
| S12 참석여부 | 인앱 폼 (이름, 신랑/신부측, 참석, 식사, 동반인, 메시지) → 자체 DB 저장 | ✅ |
| S13 마음 전하실 곳 | 아코디언 (신랑측/신부측) → 계좌번호 + 복사 버튼 + 카카오페이 | ✅ |
| S14 방명록 | 카드 목록 (이름+시간+메시지) + [작성] | ✅ |
| S15 함께한 시간 | 첫 만남(2022.12.24)부터 초 단위 실시간 카운터 | ✅ |
| S16 엔딩 | 세로 연속 커플 사진 2장 | ✅ |

### 게스트스냅 서브페이지
| 기능 | 상태 | 비고 |
|---|---|---|
| 안내 문구 + 업로드 폼 | ✅ | 혼례일 전 비활성 |

### 백엔드 API
| 기능 | 상태 | 비고 |
|---|---|---|
| 방명록 조회 (GET /api/guestbook) | ✅ | 최신순 |
| 방명록 작성 (POST /api/guestbook) | ✅ | |
| 방명록 삭제 (DELETE /api/guestbook) | ✅ | x-admin-password 헤더 |
| 게스트스냅 업로드 (POST /api/guest-snap) | ✅ | Supabase Storage |
| 참석여부 등록 (POST /api/attendance) | ✅ | 공개, 유효성 검증 |
| 참석여부 조회 (GET /api/attendance) | ✅ | 관리자 전용 |
| 관리자 인증 (POST /api/admin/auth) | ✅ | ADMIN_PASSWORD 비교 |
| 설정 조회 (GET /api/admin/config) | ✅ | wedding_config 테이블 |
| 설정 저장 (PUT /api/admin/config) | ✅ | 관리자 전용 |
| 이미지 업로드 (POST /api/admin/images) | ✅ | 관리자 전용, Supabase Storage |
| 이미지 삭제 (DELETE /api/admin/images) | ✅ | 관리자 전용 |

### 관리자 페이지 (/admin)
| 기능 | 상태 | 비고 |
|---|---|---|
| 비밀번호 보호 게이트 | ✅ | sessionStorage 임시 저장 |
| 탭 1 — 콘텐츠 관리 | ✅ | 커버·갤러리·스토리·탭 이미지 업로드/삭제/순서 변경 |
| 탭 2 — 예식 정보 | ✅ | 청첩 문구, 예식장명·주소·시간, 커플 사진 |
| 탭 3 — 인물·인터뷰 | ✅ | 양가 부모, 자녀 순서, 인터뷰 Q&A 추가/삭제, 타임라인 추가/삭제 |
| 탭 4 — 안내·계좌 | ✅ | 포토부스/주차/답례품 텍스트, 계좌·카카오페이 |
| 탭 5 — 기타 | ✅ | BGM YouTube URL, 참석여부 구글폼 URL |
| 탭 6 — 방명록 | ✅ | 목록 조회 + 삭제 |
| 저장 후 미리보기 링크 | ✅ | 저장 5초 후 자동 숨김 |
| 토스트 알림 | ✅ | 성공/실패 메시지 |
