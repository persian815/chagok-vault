## v0.1 — MVP (본식 전 완성 목표)

### 레이아웃 & 공통 UI
| 기능 | 상태 | 비고 |
|---|---|---|
| 진입 팝업 (참석여부 모달) | 📋 | 배경 딤처리, 오늘하루 안보기, X 닫기 |
| 오늘하루 안보기 (localStorage) | 📋 | |
| 배경음악 재생/정지 버튼 | 📋 | 팝업 닫기 시 자동 재생 시작 |
| 위로가기 + 공유 버튼 (우측 하단 상시) | 📋 | |
| 골드 포인트 컬러 (#C4A882) | 📋 | CSS 변수로 관리 |
| Playfair Display (영문 세리프) 폰트 | 📋 | Google Fonts |
| 스크롤 fade-in 애니메이션 | 📋 | Intersection Observer |

### 섹션
| 섹션 | 기능 | 상태 |
|---|---|---|
| S01 히어로 커버 | 전체화면 사진 + 세리프 타이포 + 날짜 | 📋 |
| S02 초대 문구 | 텍스트 + 전체너비 사진 | 📋 |
| S03 인물 소개 | 신랑/신부 2열 카드 (사진·이름·📞·생년·키워드·양가부모) | 📋 |
| S04 웨딩 인터뷰 | 버튼 → 바텀시트 모달 (Q&A, 신랑🤵/신부👰 구분) | 📋 |
| S05 예식 안내 | 날짜·시간·장소 + 커플 사진 + 9월 달력 (19일 골드 원) | 📋 |
| S06 D-Day | 전체화면 사진 오버레이 + 💗 + D-XXX + 날짜 | 📋 |
| S07 우리의 이야기 | 지그재그 타임라인 (사진↔텍스트, 중앙 세로선+골드 점) | 📋 |
| S08 갤러리 | 3열 모자이크 그리드 → 라이트박스 (스와이프, n/8, X) | 📋 |
| S09 게스트스냅 | 서브페이지 링크 버튼 | 📋 |
| S10 포토부스·주차·답례품 | 3탭 네비 (사진 + 안내 텍스트) | 📋 |
| S11 오시는 길 | 장소+📞 + 주소 + 네이버지도 iframe + [티맵\|카카오내비\|네이버지도] + 교통 안내 | 📋 |
| S12 참석여부 | 설명 문구 + 골드 버튼 (구글폼 외부 링크) | 📋 |
| S13 마음 전하실 곳 | 아코디언 (신랑측/신부측) → 계좌번호 + 복사 버튼 + 카카오페이 | 📋 |
| S14 방명록 | 카드 목록 (이름+시간+메시지+X) + [전체보기] [작성] | 📋 |
| S15 함께한 시간 | 첫 만남(2022.12.24)부터 초 단위 실시간 카운터 | 📋 |
| S16 엔딩 | 세로 연속 커플 사진 2장 | 📋 |
