# 모바일 청첩장 (weddingCard)

> 차곡 브랜드 3호 프로젝트
> 성룡 & 주영의 결혼을 알리는 감성 모바일 청첩장

---

## 한 줄 요약

섹션 16개, 방명록, 게스트스냅, 참석여부, 배경음악, 관리자 설정 페이지가 담긴 커플 전용 감성 모바일 청첩장

---

## 탄생 배경

종이 청첩장 대신 직접 개발한 감성 모바일 청첩장으로 하객에게 날짜·장소·이야기를 전달.
방명록과 게스트스냅으로 하객과 실시간 소통.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js (App Router) + TypeScript |
| Styling | Tailwind CSS + 커스텀 CSS (Keyframe 애니메이션) |
| DB | Supabase (방명록, 참석여부, 동적 설정) |
| Storage | Supabase Storage (게스트스냅, 이미지 관리) |
| 배포 | Vercel |
| 지도 | Google Maps iframe + 네이버 지도 링크 |
| BGM | YouTube URL 임베드 |

---

## 주요 기능

1. **섹션 16개** — 히어로 커버·초대문구·인물소개·인터뷰·예식안내·D-Day·우리의이야기·갤러리·게스트스냅·포토부스·오시는길·참석여부·계좌·방명록·함께한시간·엔딩
2. **진입 팝업** — 참석여부 모달 (오늘하루 안보기 + 배경음악 시작)
3. **배경음악** — YouTube URL 기반 재생/정지 토글 (팝업 닫을 때 자동 시작)
4. **갤러리 라이트박스** — 3열 그리드 → 전체화면 스와이프
5. **방명록** — Supabase DB 저장·조회·관리자 삭제
6. **게스트스냅** — 혼례일 당일부터 사진 업로드 (Supabase Storage)
7. **참석여부 직접 수집** — `wedding_attendance` 테이블 (이름, 신랑/신부측, 참석, 식사, 동반인, 메시지)
8. **관리자 페이지** (`/admin`) — 비밀번호 보호, 6탭 설정 UI
   - 콘텐츠 관리: 커버·갤러리·스토리·탭 이미지 업로드
   - 예식 정보: 청첩 문구, 예식장, 시간, 커플 사진
   - 인물·인터뷰: 양가 부모, 인터뷰 Q&A, 우리 이야기 타임라인
   - 안내·계좌: 포토부스/주차/답례품 텍스트, 계좌 정보
   - 기타: BGM URL, 참석여부 구글폼 URL
   - 방명록: 목록 조회 및 삭제
9. **WeddingConfig 동적 설정** — DB(`wedding_config`) 기반, 환경변수 없이 관리자 UI로 모든 콘텐츠 변경 가능
