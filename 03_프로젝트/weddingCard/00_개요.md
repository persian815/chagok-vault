# 모바일 청첩장 (weddingCard)

> 차곡 브랜드 3호 프로젝트
> 성룡 & 주영의 결혼을 알리는 감성 모바일 청첩장

---

## 한 줄 요약

섹션 16개, 방명록, 게스트스냅, 배경음악이 담긴 커플 전용 감성 모바일 청첩장

---

## 탄생 배경

두리(WPC) 웨딩 플래너와 함께 만드는 결혼 준비의 한 축.
종이 청첩장 대신 직접 개발한 감성 모바일 청첩장으로 하객에게 날짜·장소·이야기를 전달.
방명록과 게스트스냅으로 하객과 실시간 소통.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js (App Router) + TypeScript |
| Styling | Tailwind CSS + 커스텀 CSS (Keyframe 애니메이션) |
| DB | Supabase (방명록) |
| Storage | Supabase Storage (게스트스냅) |
| 배포 | Vercel |
| 참석 여부 | 구글 폼 외부 링크 |

---

## 주요 기능

1. **섹션 16개** — 히어로 커버·초대문구·인물소개·인터뷰·예식안내·D-Day·우리의이야기·갤러리·게스트스냅·포토부스·오시는길·참석여부·계좌·방명록·함께한시간·엔딩
2. **진입 팝업** — 참석여부 모달 (오늘하루 안보기 + 배경음악 시작)
3. **배경음악** — 재생/정지 토글 (팝업 닫을 때 자동 시작)
4. **갤러리 라이트박스** — 3열 그리드 → 전체화면 스와이프
5. **방명록** — Supabase DB 저장·조회·관리자 삭제
6. **게스트스냅** — 혼례일 당일부터 사진 업로드 (Supabase Storage)
