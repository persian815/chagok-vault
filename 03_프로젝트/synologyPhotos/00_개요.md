# SynoPhoto Manager (synologyPhotos)

> 차곡 브랜드 3호 프로젝트
> 시놀리지 NAS에 저장된 사진을 웹에서 관리·정리하는 개인용 도구

---

## 한 줄 요약

중복·유사 사진 제거, EXIF 기반 파일명 변경, 연도·월 폴더 재정리를 웹에서 한 번에 처리하는 시놀리지 포토 관리 앱

---

## 탄생 배경

아이폰 → PC → 시놀리지 포토로 연도별 수동 백업을 해오다 보니
중복 사진, 잘못된 폴더에 들어간 파일들이 쌓여 있음.
아이클라우드 동기화 불편으로 수동 방식을 유지하되, 이를 효율적으로 관리하기 위한 전용 도구 필요.

---

## 기술 스택

| 분류 | 기술 |
|---|---|
| Frontend | Next.js 15 (App Router) + TypeScript |
| Styling | Tailwind CSS + shadcn/ui |
| DB/Auth | 없음 (시놀리지 NAS 자체가 데이터 소스) |
| 시놀리지 연동 | Synology DSM REST API (SYNO.Foto.*) |
| 이미지 해싱 | sharp + imghash (pHash) |
| EXIF 파싱 | exifr |
| 세션 | iron-session |
| 배포 | 로컬 실행 우선 (npm run dev) |

---

## 주요 기능

1. **DSM 로그인** — QuickConnect URL + 계정으로 시놀리지 세션 인증
2. **사진 그리드** — 폴더별 썸네일 그리드 보기
3. **중복 제거** — MD5 해시 기반 완전 중복 감지 + 선택 삭제
4. **유사 사진** — pHash 기반 연속 촬영·유사 구도 감지 + 선택 삭제
5. **파일명 변경** — EXIF 촬영일 기반 `YYYYMMDD_HHMMSS.jpg` 자동 변경
6. **폴더 정리** — EXIF 촬영일 → `/개인공간/YYYY/YYYY-MM` 자동 재배치
